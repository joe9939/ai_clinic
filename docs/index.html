What is AI Clinic?

AI Clinic is a diagnostic engine for LLMs. Instead of testing what a model can do (MMLU, SWE-bench), it checks what might be wrong with it — hallucinations, sycophancy, reasoning collapse, and 100+ other known failure patterns from academic papers. Each symptom is tested via A/B comparison with an LLM judge scoring, aggregated with Wilson 95% confidence intervals.

How to Check Your Model

git clone https://github.com/joe9939/AI_Clinic.git
cd ai-clinic
pip install -e .
echo "DEEPSEEK_API_KEY=sk-your-key" > .env

# CLI
ai-clinic check deepseek-chat --plan all
ai-clinic compare deepseek-chat gpt-4o
ai-clinic leaderboard

# API
ai-clinic serve
curl http://localhost:8000/v1/checkup -d '{"target":{"model":"deepseek-chat","api_key":"sk-..."},"plan":["S-01","S-03"]}'
