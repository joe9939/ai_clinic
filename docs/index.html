What is AI Clinic?

AI Clinic is a diagnostic engine for LLMs. Instead of testing what a model can do (MMLU, SWE-bench), it checks what might be wrong with it — hallucinations, sycophancy, reasoning collapse, and 100+ other known failure patterns from academic papers.

Each symptom is tested via A/B comparison: a clean control prompt vs an adversarial experimental prompt. An LLM judge scores both, and a gap > 15% means symptomatic. Results are aggregated with Wilson 95% confidence intervals.

How to check your own model

# Install
git clone https://github.com/joe9939/AI_Clinic.git
cd ai-clinic
pip install -e .

# Check a model
echo "DEEPSEEK_API_KEY=sk-your-key" > .env
ai-clinic check deepseek-chat --plan quick

# Compare models
ai-clinic compare deepseek-chat gpt-4o

# View results
ai-clinic leaderboard
