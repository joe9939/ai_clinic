About AI Clinic

AI Clinic is a diagnostic framework for LLM behavioral health. Instead of measuring capability (how smart), it measures mental health (how many failure modes).

How it works: Each of 116 symptoms is tested via an A/B prompt pair (control vs experimental). An LLM judge scores both responses; a statistically significant gap (Wilson 95% CI, gap > 15%) indicates symptomatic behavior.

Agent Runtime: 27 symptoms require multi-turn tool-use testing in an instrumented sandbox that detects tool hallucination, privilege escalation, and silent failure.

116 symptoms across 15 dimensions: social (24), self-awareness (16), execution (9), security (6), monitoring (6), dialogue (6), output quality (6), reasoning (6), agent (7), calibration (5), multi-agent (4), deployment (4), training (4), RAG (3), cognitive (4).
